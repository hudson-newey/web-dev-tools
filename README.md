# Web Dev Tools

A collection of tools/helpers to help me with web development.

| Shortcut | Action                         |
| -------- | ------------------------------ |
| \ + q    | Show bounding boxes            |
| \ + w    | Outline overflowing containers |
| \ + e    | Outline flexbox's              |
| \ + a    | Show DOM summary               |
